# Репозиторий для pull request

* В своём аккаунте на *GitHub* создать копию репозитория **ValerieSharandina/ForHomeWork_Git** с помощью кнопки "Fork".
* В своей копии репозитория нажимаем кнопку Code и копируем ссылку HTTPS.
* Открыть новую папку в VSCode.
* Клонировать копию репозитория на локальный компьютер командой git clone Ctrl+V.
* Перейти в новую появившуюся папку командой cd название_папки.
* **Создать новую ветку командой git branch**
* Перейти в новую ветку командой git checkout название_ветки.
* Добавить файл с инструкцией в новую ветку.
* Дополнить инструкцию разделами по работе с удалёнными репозиториями, pull request.
* Зафиксировать изменения (коммиты).
* Отправить изменения на свой GitHub командой git push.
* На сайте GitHub выполнить Pull request и все изменения отправятся к первоначальному владельцу репозитория.
